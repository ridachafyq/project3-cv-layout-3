# Project 3 — CV Layout 3

A **CSS-only interactive CV** to practice modern UI patterns and accessibility.

## Features

- Scroll-snapping sections with active highlighting (`:target`)  
- Circular “donut” meters using `conic-gradient`  
- Accessible `<details>` / `<summary>` accordions  
- Glassmorphism cards with subtle hover lift  
- Auto dark mode & reduced-motion support
